Отчет по лабораторной работе:

1. Программа собрана логично и по модулям. Главная функция `gen_bin_tree` отдельно считает ветки дерева через `left_branch` и `right_branch`. Для сравнения, как лучше хранить дерево, попробовал разные структуры данных: `gen_bin_tree_list` и `gen_bin_tree_tuple`.

2. Функиция `gen_bin_tree()` умеет гибко работать с данными и проверять их:

- Не даст ввести отрицательную высоту дерева (выдаст `ValueError`).
- Если свои функции для веток не указаны, возьмет стандартные `l_b` и `r_b`.
- Можно задавать любые формулы для веток через `lambda`-функции или обычные функции.
- Числа, которые получаются для ключей словаря, преобразует в текст.

3. В программе есть разные варианты использования:

- Можно построить стандартное дерево по заданию (root=13, height=3).
- Можно задать свои функции для левой и правой ветки через параметры `l_b` и `r_b`.
- Деревья могут быть разной высоты (0, 1, 2 и больше).
- Можно менять значение корня дерева.

4. Тесты проверяют:

- Как строятся деревья высотой 0, 1 и 2.
- Как работает программа со стандартными настройками (высота 3, корень 13).
- Функции разных типов (сложение, умножение, возведение в степень).
- Lambda-функции, чтобы гибко задавать формулы для веток.
- Разные значения для корня дерева.
- Правильно ли обрабатываются ошибки (отрицательная высота).
- Как работают альтернативные структуры данных (список, кортеж).

5. Код оформлен по правилам PEP-257:

- У каждой функции есть описание, что она делает, какие параметры принимает и что возвращает.
- Чтобы код было легче понимать, использованы аннотации типов (`Callable`, `Optional`).
- В комментариях объясняются важные моменты в алгоритмах.

6. В исследовательской части:

- Показал, что удобно передавать функции для веток как параметры.
- Доказал, что программа работает и со старыми параметрами из задания.

-  Можно добавлять новые функции для веток, не меняя основной код.
-  Каждая функция отвечает только за что-то одно.
-  Дерево строится с помощью рекурсии, что естественно для такой структуры.

8. Решение показывает, что программу можно использовать для построения разных бинарных деревьев с разными способами ветвления.
